# 서울, 미제사건 (Seoul Cold Case)

보드게임 *Deception: Murder in Hong Kong* 한국어판을 기반으로 한 모바일 웹 멀티플레이어 게임.

## MVP 1차 — 동기화 검증용

이 버전은 다음만 구현:
- 방 생성 / 입장 (4자리 코드)
- 참가자 목록 실시간 동기화
- 카드 분배 + 역할 배정 (사인 100장, 단서 238장 풀)
- 수사관 폰 화면 (자기 카드 8장 + 역할)

다음 버전에서 추가 예정:
- 마커 배치 (큰 화면)
- 추리 시도
- 결과 화면

## 기술 스택
- React 18 + Vite
- Firebase Realtime Database + Anonymous Auth
- Vercel 배포

## 라이선스
원작 게임의 룰을 한국어로 재해석한 팬 프로젝트.
